DE-REVERB & ESSER (Mastering Tool v1.2) 사용 설명서

이 앱은 브라우저 내에서 100% 로컬로 구동되는 전문 오디오 마스터링 툴입니다. 별도의 설치나 서버 전송 없이 AI급 오디오 보정 기능을 즉시 사용할 수 있습니다.

1. 시작하기
	a. 파일 불러오기: 우측 상단의 [IMPORT AUDIO] 버튼을 눌러 보정할 오디오 파일(WAV, MP3, M4A 등)을 업로드합니다.
	b. 재생 및 모니터링: 하단의 [PLAY] 버튼을 눌러 재생하면서 실시간으로 소리가 어떻게 변하는지 모니터링합니다.
		* Dry Analysis: 원본 소리를 분석합니다.
		* Processed: 보정된 소리를 출력합니다. (두 버튼을 번갈아 누르며 전후를 비교해 보세요.)

2. 주요 기능 설정

	① 자동 잔향 억제 (Auto De-Reverb)
		* Gate Threshold: 소리가 차단되는 기준점을 설정합니다. 값이 높을수록 배경 노이즈와 잔향이 강하게 차단됩니다.
		* Reduction Ratio: 잔향을 얼마나 줄일지 결정합니다. 너무 높으면 목소리가 부자연스러울 수 있으니 적절히 조절하세요.
		* [Smart Cleaning] 버튼: 클릭 한 번으로 최적의 잔향 제거 설정을 자동으로 적용합니다.
	② 자동 치찰음 억제 (Auto De-Esser)
		* Target Frequency: 억제할 치찰음(ㅅ, ㅈ, ㅊ 등)의 주파수 대역을 설정합니다. (보통 5000Hz~8000Hz 사이)
		* Strength Level (1~5): 치찰음 제거 강도를 선택합니다. 마우스를 올리면 각 단계별 상세 가이드를 볼 수 있습니다. (3단계 권장)
	③ 음량 정규화 (Normalization)
		* -14 LUFS: 유튜브, 스포티파이 등 스트리밍 플랫폼 표준 음량입니다.
		* -16 LUFS: 팟캐스트 및 방송 표준 음량입니다.
		클릭 시 해당 표준에 맞춰 전체 음량이 자동으로 평준화됩니다.

3. 정밀 조절 기능 (Expert Control)
	* 파형 노드 제어: 상단 파형 그래프에 있는 **파란색 노드(L3)**를 마우스로 직접 드래그하여 특정 시점의 치찰음을 수동으로 억제할 수 있습니다.
	* 실시간 스펙트로그램: 하단 시각 가이드를 통해 고음역대 주파수가 깎이는 양상을 눈으로 확인하며 작업하세요.

4. 내보내기 (Export)
	* 작업이 완료되면 Export Configuration에서 파일 형식(WAV 또는 MP3 320k)을 선택합니다.
	* [MASTER EXPORT] 버튼을 누릅니다.
	브라우저 내에서 즉시 믹싱이 시작되며, 완료 시 보정된 파일이 자동으로 다운로드됩니다.

💡 전문가 팁 (Pro Tip)
	* 반복 작업: 낮은 강도로 치찰음을 제거하고 결과물을 다운로드한 뒤, 그 파일을 다시 Import하여 다시 반복적으로 치찰음 억제를 적용하면 훨씬 더 깔끔한 고품질 사운드를 얻을 수 있습니다.
	* 100% 프라이버시 보호: 이 앱은 모든 처리를 사용자의 컴퓨터 안에서만 수행합니다. 소중한 음원 데이터가 외부 서버로 절대 유출되지 않으니 안심하고 사용하세요.
